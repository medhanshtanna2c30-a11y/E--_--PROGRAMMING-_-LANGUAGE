E++ Studio

E++ Studio is a simple browser-based coding studio for the E++ language.

What it can do

- Write E++ code in an editor
- Run code and show output
- Learn with Explain Mode
- Use variables with "set"
- Print text and values with "show"
- Make decisions with "if", "else", and "end"
- Repeat code with "repeat" and "end"
- Save code automatically in the browser
- Switch between light and dark themes

E++ Example

set score = 10

if score >= 10
  show "Level unlocked"
else
  show "Keep going"
end

Project Files

- "index.html" — the E++ Studio page
- "style.css" — the design and layout
- "script.js" — the E++ runner and buttons
- "README.md" — project information
